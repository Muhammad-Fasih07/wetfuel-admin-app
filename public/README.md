# Public Assets

Place your logo file here:

- **`logo.png`** — Main logo (recommended: 200×200px, transparent background)
- **`logo.svg`** — SVG version (current placeholder)
- **`favicon.ico`** — Browser tab icon

Files in this folder are served at the root URL.  
e.g. `public/logo.png` → accessible at `http://localhost:3000/logo.png`

Replace `logo.svg` with your own logo file and update the `src` in:
- `components/auth/BrandMark.tsx`
